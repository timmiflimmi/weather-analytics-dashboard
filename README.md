# Weather Analytics Project

Analyse von Wettertrends mit der OpenWeatherMap API.

## Setup
1. `pip install -r requirements.txt`
2. API-Key von openweathermap.org holen
3. `.env` Datei erstellen (siehe `.env.example`)
